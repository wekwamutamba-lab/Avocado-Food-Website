# 🥑 ZimAvo Food Website

A modern avocado food website built with React.

## Features

- Multi-page navigation with React Router
- Reusable React components
- Product cards using props
- Dynamic product rendering with `.map()`
- Interactive buttons using `onClick`
- Responsive styling with CSS

## Technologies

- React
- JavaScript
- React Router
- CSS
- Vite

## Run Locally

```bash
npm install
npm run dev
